moderator

Useful commands for managing your server
< > = required | [ ] = optional | aliases = second name

ban <member> [reason]

Bans a user from the current server.

clear <amount>

Clears x amounts of messages from a channel.

find

Finds a user within your search term

kick <member> [reason]

Kicks a member from the server.

lock

Lockdowns a channel for everyone to send messages.

massban <reason> [members...]

Mass bans multiple members from the server.

mute <member> [reason]

Mutes a user from the current server.

nickname <member> [name]
aliases :: nick

Change nickname of a member.

prune

Removes messages from the current server.

unban <member> [reason]

Unbans a user from the current server.

unlock

Unlocks a channel for everyone to send messages in.

unmute <member> [reason]

Unmutes a user from the current server.

warn <member> <reason>

Warn a member.
