Create art via commands
< > = required | [ ] = optional | aliases = second name

ascii <text>

Convert text to art

asciifont <text> <font>

Convert text to art with chooseable font

asciifonts

List all ascii fonts to use for asciifont

bigtext <text>
aliases: big

Enlarge any text. Actually just ascii with the 'big' font.
