infocommands

Commands to get info about many things
< > = required | [ ] = optional | aliases = second name

about
aliases :: botstats, stats

Tells you information about the bot itself.

avatar [member]

Get the avatar of you or someone.

botprefix

Shows the bot's prefixs

changelog
aliases :: changes, whatsnew

See what's new, changed, fixed or removed

channelinfo [channel]

Shows info about any channel.

color <hex>
aliases :: colour

View the colors HEX details. Random color when given `random`.

count

See me flexing :D

dblbot <member>

Get a bot's info from discordbots.org

game [member]

Tells you what someone is playing in a nice embed.

joinedat [member]

Check when a user joined the current server

memberstats
aliases :: ms

Check how many are online, offline, idle or dnd.

moderators
aliases :: mods

Lists all the moderators of the current server

perms [user]

Check what perms you have and dont.

role

role info commands.

server
aliases :: guild

Server info commands.

source

Check out the source code <3

steam <id>

Find users on steam.

userinfo [member]

Shows info about you or a member.

weather <location>

Get the weather!

ytchannel BETA <channelname>
aliases :: youtubechannel, ych, yt

Get info about a YT Channel.
